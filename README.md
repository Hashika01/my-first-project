# My First Project 🚀

A simple web project built using HTML, CSS, and JavaScript and deployed using GitHub Pages.

## 🌐 Live Demo

https://hashika01.github.io/my-first-project/

## 📌 About the Project

This project was created to learn the fundamentals of web development and the complete deployment workflow using GitHub and GitHub Pages.

Through this project, I learned how to:

* Create and structure web pages using HTML
* Style websites using CSS
* Add interactivity using JavaScript
* Use Git for version control
* Push code to GitHub repositories
* Deploy a website using GitHub Pages

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Git
* GitHub
* GitHub Pages

## 📂 Project Structure

```text
MyFirstProject/
│

```

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/Hashika01/my-first-project.git
```

2. Open the project folder

```bash
cd my-first-project
```

3. Open `index.html` in your browser or use Live Server in VS Code.
